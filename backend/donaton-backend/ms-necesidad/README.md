# Microservicio: Necesidades (ms-necesidad)

Gestión y seguimiento de las necesidades de los beneficiarios.

## Patrones Utilizados
*   **DTO Pattern:** Uso de objetos de transferencia de datos (NecesidadDTO) para no exponer las entidades de base de datos directamente al cliente.
