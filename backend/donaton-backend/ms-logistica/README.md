# Microservicio: Logística (ms-logistica)

Gestión de centros de acopio y asignación de logística para las donaciones.

## Patrones Utilizados
*   **Service Layer:** Separación clara entre el Controlador (LogisticaController) y la lógica de negocio (LogisticaService).

## Instalación y Ejecución
1. Asegurarse de tener la BD arriba.
2. Ejecutar: mvn spring-boot:run
*Puerto de ejecución: 8083*
